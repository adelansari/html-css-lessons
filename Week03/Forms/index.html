Radio buttons allow you to use one option.

type="file" defines a file-select field and a "Browse" button for file uploads.
type="image" defines an image as a submit button.
type="month" allows the user to select a month and year
Text Area The size of a text area can be specified by the cols and rows attributes, or even better; through CSS' height and width properties.
type="number" defines a numeric input field.
type="password"
type="range" defines a control for entering a number whose exact value is not important (like a slider control). Default range is 0 to 100
Select example By default this creates a dropdown that allows the user one choice. Adding size and multiple attributes will allow the user to pick many.
